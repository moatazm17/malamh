## IMPORTANT: This prompt is ONLY about visual design, UI, UX, animations, and styling. Do NOT change any backend logic, API routes, database schema, or functionality. Only redesign how everything LOOKS and FEELS.

## Global Design System

### Aesthetic Direction
Dark, cinematic, high-end. NOT generic startup. NOT "AI slop." Think Linear.app + Stripe + Christopher Nolan movie poster. This website protects something valuable — it should FEEL like it.

### Color System (use as CSS variables everywhere)
```css
:root {
  --bg-void: #050507;
  --bg-primary: #0a0a0f;
  --bg-elevated: #111118;
  --bg-hover: #1a1a24;
  --border-subtle: #1e1e2e;
  --border-glow: #2a2a4a;
  
  --text-primary: #f0f0f5;
  --text-secondary: #8888a0;
  --text-muted: #555566;
  
  --accent-blue: #4d7cff;
  --accent-blue-glow: rgba(77, 124, 255, 0.15);
  --accent-red: #ff4d5e;
  --accent-red-glow: rgba(255, 77, 94, 0.12);
  --accent-green: #00d48a;
  --accent-green-glow: rgba(0, 212, 138, 0.12);
  --accent-amber: #ffb020;
  --accent-amber-glow: rgba(255, 176, 32, 0.12);
}
```

### Typography
- Display/Hero: "Cabinet Grotesk" from https://api.fontshare.com/v2/css?f[]=cabinet-grotesk@700,800,900&display=swap — weights 700, 800, 900
- Body: "Satoshi" from https://api.fontshare.com/v2/css?f[]=satoshi@400,500,600,700&display=swap — weights 400, 500, 600
- Mono/Code: "JetBrains Mono" from Google Fonts — for API code snippets
- Arabic (ملامح brand only): "IBM Plex Sans Arabic" from Google Fonts
- NEVER use Inter, Roboto, Arial, system-ui, or any generic font anywhere

### Global Effects (apply to ALL pages)
- Noise texture overlay on body: `background-image: url("data:image/svg+xml,...")` with opacity 0.03, creates film grain feel
- All cards: `background: rgba(17, 17, 24, 0.6); backdrop-filter: blur(20px); border: 1px solid var(--border-subtle);` with hover → `border-color: var(--border-glow); transform: translateY(-2px);`
- All buttons primary: `background: var(--accent-blue); box-shadow: 0 0 30px var(--accent-blue-glow);` with hover → shadow expands to 50px
- All buttons ghost/secondary: `background: transparent; border: 1px solid var(--border-subtle); color: var(--text-secondary);` with hover → `border-color: var(--accent-blue); color: var(--text-primary);`
- All page transitions: elements fade in + slide up 20px on mount, staggered 0.05s between siblings
- Scroll-triggered animations: use Intersection Observer, elements start opacity:0 translateY:30px, animate to opacity:1 translateY:0 with ease-out 0.6s
- Input fields: `background: var(--bg-primary); border: 1px solid var(--border-subtle); color: var(--text-primary);` with focus → `border-color: var(--accent-blue); box-shadow: 0 0 20px var(--accent-blue-glow);`
- Status badges: BLOCKED = red bg with red glow, TOKEN_REQUIRED = amber bg with amber glow, OPEN = green bg with green glow. All with `border-radius: 9999px; padding: 4px 12px; font-size: 0.75rem; font-weight: 600;`
- Tables: no visible borders. Rows separated by subtle 1px border-bottom in border-subtle. Hover row → bg-hover. Header text → text-muted, uppercase, letter-spacing 0.1em, font-size 0.7rem.
- Modal/Dialog: centered, glass card, dark overlay with blur behind it

## Responsive Breakpoints

- Desktop: > 1024px (sidebar visible, multi-column layouts)
- Tablet: 768px - 1024px (sidebar collapses to hamburger, 2-column → 1-column)
- Mobile: < 768px (everything stacked, full-width cards, bottom nav optional)

## Animation Performance Rules

- Use `transform` and `opacity` ONLY for animations (never animate width/height/top/left)
- Use `will-change: transform, opacity` on animated elements
- Respect `prefers-reduced-motion` — disable all motion animations if set
- Intersection Observer for scroll triggers — disconnect after first trigger (animate once, not repeatedly)

## Accessibility

- All interactive elements must have visible focus states (accent-blue ring)
- Color is never the ONLY indicator — always pair with icon or text
- Minimum contrast ratio 4.5:1 for body text, 3:1 for large text
- All images have alt text
- Buttons have aria-labels when icon-only
